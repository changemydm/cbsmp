## 💡 1. Блок света

<div class="craft-item">

### 📝 Описание
Невидимый блок, который производит **15-й уровень света**. Идеально подходит для строителей — позволяет создавать освещенные пространства без видимых источников света.

### 🛠️ Использование
| Действие | Описание |
|:---------|:---------|
| **Установка** | Размещается как обычный блок |
| **Освещение** | Производит максимальный (15) уровень света |
| **Демонтаж** | Чтобы сломать поставленный блок света, возьмите **другой блок света** в руку и наведите на поставленный |

## 🖼️ 2. Невидимая рамка

<div class="craft-item">

### 📝 Описание
Рамка для предметов с эффектом невидимости. Предмет в такой рамке будет "парить" в воздухе.

### 🛠️ Использование
| Действие | Описание |
|:---------|:---------|
| **Установка** | Размещается как обычная рамка |
| **Эффект** | Рамка становится невидимой, предмет остается видимым |
| **Применение** | Идеально для декораций, витрин, парящих предметов |

## 🔧 3. Палка отладки (Debug Stick)

<div class="craft-item">

### 📝 Описание
Специальный инструмент для видоизменения блоков. Позволяет тонко настраивать внешний вид различных строительных блоков.

### 🛠️ Использование
| Действие | Описание |
|:---------|:---------|
| **Поворот** | Изменяет направление ступенек, плит, блоков |
| **Трансформация** | Меняет вид ограждений, заборов, стен |
| **ЛКМ** | Изменяет состояние/положение блока |
| **ПКМ** | Переключает режим редактирования |

### ✨ Возможности
| Блок | Что можно изменить |
|:-----|:-------------------|
| **Ступеньки** | Направление, форма (верх/низ) |
| **Заборы** | Соединения с соседними блоками |
| **Плиты** | Положение (верх/низ) |
| **Кровать** | Направление изголовья |
| **Двери/Люки** | Направление открывания |
