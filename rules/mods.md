# 🚫 CubeSMP — Запрещенные модификации

*Данный список распространяется на всех игроков без исключения. Использование любого из перечисленного ниже софта является нарушением правил сервера.*

---

</div>

## ⚖️ Общие положения

| Пункт | Описание |
|:------|:---------|
| **1.1** | Запрещено использование чит-клиентов, модов, текстурпаков и любого другого софта, который дает преимущество над другими игроками |
| **1.2** | Запрещено **хранение** запрещенных модификаций в папке с игрой во время игры на сервере |
| **1.3** | Наказание за нарушение — **бан по усмотрению администрации**. Признание нарушения смягчает наказание |
| **1.4** | Администрация вправе расширять данный список без предварительного уведомления |

> 💡 *Если вы не уверены, разрешен ли конкретный мод — обратитесь к администрации*

## 📋 Полный список запрещенных модификаций


### 1. XRay ресурспаки и моды

| Название | Описание |
|:---------|:---------|
| **XRay текстурпаки** | Ресурспаки, делающие руды видимыми сквозь стены |
| **XRay моды** | Любые моды, показывающие руды сквозь текстуры |
| **Wallhack** | Моды, убирающие стены или делающие их прозрачными |
| **FullBright моды** | Моды, убирающие затемнение в пещерах |
| **Gamma моды** | Моды, повышающие яркость |

> ❗ **Важно:** Любой мод или ресурспак, который позволяет видеть руды сквозь блоки, считается XRay и **строго запрещен**

### 10. Запрещенные функции в разрешенных модах

Многие разрешенные моды (OptiFine, Sodium, Lithium и др.) имеют **функции, которые запрещены**:

| Мод | Запрещенные функции |
|:----|:-------------------|
| **FullBright моды** | Любые функции, повышающие яркость выше ванильной |
| **Gamma моды** | Любые функции, повышающие гамму выше ванильной |
| **Shaders** | Шейдеры, убирающие листву или делающие стены прозрачными |

## ⚠️ Важное предупреждение

| Пункт | Описание |
|:------|:---------|
| **1** | Администрация не обязана доказывать наличие запрещенных модификаций |
| **2** | Решение о том, является ли мод запрещенным, принимает администрация |
| **3** | Использование ВПН/прокси для обхода бана ведет к блокировке всех связанных аккаунтов |
| **4** | Хранение запрещенных модификаций в папке с игрой приравнивается к их использованию |
