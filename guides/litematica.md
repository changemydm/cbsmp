Litematica — мод, который выводит строительство на новый уровень. Загружай схему постройки и строй по голограмме

---

# 📋 Общая информация

| Параметр              | Описание                                          |
| :-------------------- | :------------------------------------------------ |
| **Назначение**        | Строительство по визуальному чертежу (голограмме) |
| **Тип мода**          | Клиентский (устанавливается только игроком)       |
| **Статус на сервере** | ✅ **РАЗРЕШЕН** (EasyPlace Mode так же разрешён)  |
| **Важно**             | ❌ **Мод-принтер (автострой) ЗАПРЕЩЁН**           |

> 💡 _Мод идеально подходит для переноса готовых проектов, совместного строительства и масштабных построек_

---

# ⚙️ 1. Установка мода

## 📦 Необходимые компоненты

| Компонент      | Описание                         | Где скачать                    |
| :------------- | :------------------------------- | :----------------------------- |
| **Fabric**     | Загрузчик модов (обязателен)     | fabricmc.net                   |
| **Fabric API** | Базовый API для модов            | CurseForge / Modrinth          |
| **MaLiLib**    | Библиотека для работы Litematica | GitHub / CurseForge / Modrinth |
| **Litematica** | Сам мод                          | GitHub / CurseForge / Modrinth |

## 🔧 Пошаговая установка

|  Шаг  | Действие                                                            |
| :---: | :------------------------------------------------------------------ |
| **1** | Установите **Fabric** через официальный установщик                  |
| **2** | Запустите Minecraft один раз с Fabric для создания папки `mods`     |
| **3** | Скачайте **Fabric API**, **MaLiLib** и **Litematica** нужной версии |
| **4** | Переместите все `.jar` файлы в папку `.minecraft/mods`              |
| **5** | Запустите игру с профилем Fabric                                    |

> ✅ _Готово! Если в меню паузы появился пункт "Mods" и там есть Litematica — установка прошла успешно_

# 🎮 2. Основы использования

## ⌨️ Горячие клавиши (по умолчанию)

|         Клавиша         | Функция                                      |
| :---------------------: | :------------------------------------------- |
|          **M**          | Открыть главное меню мода                    |
|    **Ctrl + колесо**    | Переключение режимов инструмента             |
|      **ЛКМ / ПКМ**      | Выделение углов области (в режиме выделения) |
| **Page Up / Page Down** | Переключение слоёв                           |
|  **M + Page Up/Down**   | Включение режима послойного строительства    |

> 💡 \*Рекомендуется настроить клавиши под себя в меню **Configuration → Hotkeys\***

# 📂 3. Загрузка схемы

## 🔄 Подготовка

|  Шаг  | Действие                                                           |
| :---: | :----------------------------------------------------------------- |
| **1** | Найдите файл схемы (расширение **`.litematic`** /**`.schematic`**) |
| **2** | Поместите его в папку `.minecraft/schematics`                      |
| **3** | Если папки нет — создайте вручную                                  |

## 🖱️ Размещение в мире

|  Шаг  | Действие                                             |
| :---: | :--------------------------------------------------- |
| **1** | Нажмите **M**, чтобы открыть меню                    |
| **2** | Выберите **"Load Schematics"** (Загрузить схему)     |
| **3** | Выберите нужную схему и нажмите **"Load Schematic"** |
| **4** | Голограмма появится на месте игрока                  |

## 🎯 Точное позиционирование

| Действие                            | Описание                              |
| :---------------------------------- | :------------------------------------ |
| **Меню → Schematic Placements**     | Управление размещёнными схемами       |
| **Configure → Placement Origin**    | Ручная настройка координат и поворота |
| **Инструмент (по умолчанию палка)** | Визуальное перемещение схемы          |

# 🏗️ 4. Строительство по схеме

## 🎨 Цветовая индикация блоков

|       Цвет       | Значение                                        |
| :--------------: | :---------------------------------------------- |
|  🔵 **Голубой**  | Пропущенные (не поставленные) блоки             |
|  🩷 **Розовый**  | Лишние блоки (отсутствуют в схеме)              |
| 🟠 **Оранжевый** | Ошибочное состояние (неправильный поворот, тип) |
|  🔴 **Красный**  | Ошибочные блоки                                 |

## 📐 Режимы строительства

| Режим         | Активация         | Описание                                             |
| :------------ | :---------------- | :--------------------------------------------------- |
| **Послойный** | M + Page Up/Down  | Схема будет отображаться послойно (по координатам Y) |
| **EasyPlace** | Назначить клавишу | Быстрая установка блоков зажатием ПКМ                |

## ⚡ EasyPlace Mode

**Как включить:**

1. Откройте **Configuration → Hotkeys**
2. Найдите **easyPlaceToggle** и назначьте удобную клавишу
3. Нажмите эту клавишу в игре для активации
4. Найдите **easyPlaceUseKey** и назначьте удобную клавишу
5. Зажмите кнопку, указанную в **easyPlaceUseKey** и водите по голограмме — блоки будут ставиться автоматически

> ⚠️ _EasyPlace Mode выглядит со стороны как подозрительно быстрое строительство, но это **разрешённая** функция_

# 📊 5. Полезные инструменты

## 📝 Список материалов

|  Шаг  | Действие                                  |
| :---: | :---------------------------------------- |
| **1** | Загрузите схему                           |
| **2** | В меню схемы выберите **"Material List"** |
| **3** | Нажмите **"Refresh"** для расчёта         |
| **4** | Получите полный список нужных блоков      |

## ✅ Проверка завершённости

|  Шаг  | Действие                                       |
| :---: | :--------------------------------------------- |
| **1** | В меню схемы выберите **"Schematic Verifier"** |
| **2** | Нажмите **"Start Verification"**               |
| **3** | Мод подсветит все пропущенные блоки            |

# 🎨 6. Создание своих схем

## 📌 Выделение области

|  Шаг  | Действие                                                 |
| :---: | :------------------------------------------------------- |
| **1** | Возьмите инструмент (по умолчанию палка)                 |
| **2** | Переключитесь в режим **Area Selection** (Ctrl + колесо) |
| **3** | **ПКМ** — первый угол, **ЛКМ** — второй угол             |
| **4** | Выделенная область подсветится                           |

## 💾 Сохранение схемы

|  Шаг  | Действие                                        |
| :---: | :---------------------------------------------- |
| **1** | Нажмите **M** → **Area Editor**                 |
| **2** | Проверьте координаты углов                      |
| **3** | Введите название и нажмите **"Save Schematic"** |
| **4** | Схема сохранится в папку `schematics`           |

# ❓ 8. Частые вопросы

| Вопрос                                | Ответ                                           |
| :------------------------------------ | :---------------------------------------------- |
| **Не открывается меню по M**          | Проверьте конфликт клавиш с другими модами      |
| **Не видно голограмму**               | Нажмите **M + R** (включить рендер)             |
| **Можно ли использовать на сервере?** | Да, если разрешено правилами                    |
| **Почему в списке материалов пусто?** | Нажмите **Refresh** и подождите загрузки чанков |
| **Как повернуть схему?**              | В меню **Configure** → раздел **Rotation**      |
