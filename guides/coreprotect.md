## 📋 Общая информация

**CoreProtect** —  плагин для логирования и анти-грифства, который записывает все действия игроков на сервере. 

### Что логируется

| Категория | Действия |
|:----------|:---------|
| **Блоки** | Установка, разрушение, взаимодействие |
| **Контейнеры** | Сундуки, печи, раздатчики (что взяли/положили) |
| **Сущности** | Убийство мобов, рамки, взаимодействия |

## 🎮 1. Команды для игроков

Обычным игрокам доступен только базовый инструмент расследования.

### 🔍 Режим инспектора

| Команда | Описание |
|:--------|:---------|
| **/co i** или **/co inspect** | Включить/выключить режим расследования |

### 📝 Как использовать

| Шаг | Действие | Результат |
|:---:|:---------|:-----------|
| **1** | Введите `/co i` | В чате появится: *"CoreProtect - Режим инспектора включён"* |
| **2** | **ЛКМ** по блоку | Покажет, кто **поставил** этот блок |
| **3** | **ПКМ** по блоку | Покажет, кто **сломал** или **взаимодействовал** с блоком |
| **4** | Повторный `/co i` | Выключить режим |

### 📊 Пример вывода
```
8.87/m назад + Player1 поставил stone
2.32/d назад - Player2 сломал chest
1.44/h назад - Player2 забрал 64 diamond из сундука
```
