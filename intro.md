## 🌟 О сервере

**CubeSMP** — это уютный Vanilla-plus сервер с упором на сообщество, экономику и творчество. Мы стремимся создать дружелюбную атмосферу, где каждый игрок может найти занятие по душе: от строительства и фермерства до торговли и участия в ивентах.

| Параметр | Значение |
|:---------|:---------|
| **Версия** | 1.21.10 |
| **IP-адрес** | `cubesmp.xyz` |
| **Экономика** | Алмазная Руда (АР) |
| **Главный язык** | Русский |

## 📚 Основные разделы

<div class="cards">

### [📜 Правила сервера](rules/rules.md)
Обязательны к прочтению для всех игроков. Здесь собраны все нормы поведения, правила экономики, гриферства и использования модификаций.

### [🎮 Кастомные команды](commands/commands.md)
Удобные команды для общения, развлечений и получения информации. Магический шар, кости, личные сообщения и многое другое.

### [🚫 Запрещенные модификации](rules/mods.md)
Полный список запрещенных читов, модов, ресурспаков и функций. Ознакомьтесь, чтобы избежать недоразумений.

### [📦 Кастомные крафты](crafts/crafts.md)
Уникальные предметы, недоступные в ванильном выживании: блоки света, невидимые рамки, палка отладки и другие.

### [📐 Гайд по Litematica](guides/litematica.md)
Подробное руководство по использованию мода Litematica для комфортного строительства по схемам.

## 📊 Полезные команды

| Команда | Описание |
|:--------|:---------|
| **/online [first \| local \| total] `<игрок>`** | Информация об онлайне игрока |
| **/ping `<игрок>`** | Задержка игрока |
| **/toponline** | Топ по наигранному времени |
| **/mail** | Написать письмо оффлайн-игроку |
| **/tell /msg /r** | Личные сообщения |
| **/ball** | Задать вопрос магическому шару |
| **/coin /dice** | Случайные числа |
